# 🛠️ FixIT

FixIT es una aplicación web para la gestión rápida de incidencias y sugerencias dentro de empresas. Permite a los usuarios reportar problemas o proponer mejoras, mientras que los administradores pueden gestionar y resolver estos tickets eficientemente.

## 🚀 Características

- 📋 Creación de tickets de incidencias y sugerencias.
- 👥 Roles de usuario: empleados y administradores.
- 🔔 Notificaciones de actualizaciones en los tickets.
- 🗂️ Panel de administración para seguimiento y resolución.
- 📊 Estadísticas básicas de gestión.

## 🖥️ Tecnologías Utilizadas

- **Framework:** Vite
- **Frontend:** React
- **Backend:** Supabase
- **Base de datos:** Supabase
- **Autenticación:** Supabase Auth
- **Manejo de dependencias:** Composer, NPM
